# Requirement
 
* Python 3.7.3
* pillow
 
# Installation
 
Install Pillow with pip command.
 
```bash
pip install pillow
```
 
# Usage
 
Please create python code named "main.py".
And copy &amp; paste [this code](https://raw.githubusercontent.com/shioda999/complementary_afterimage/main/main.py).
 
Run "main.py"
 
```bash
python main.py
```
